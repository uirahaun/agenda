# Aplicação construída em Arquitetura Hexagonal
## Author - Uirá Haun

## Tecnologias utilizadas
- Spring Data
- Spring Test
- PostgreSQL 10
- IntelliJ

## Descrição

### Métodos de consulta HTTP
- POST
- GET
- PUT
- DELETE
